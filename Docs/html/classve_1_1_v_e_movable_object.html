Detailed Description

Represents any object that can be put into a scene.

VEMovableObject represents any object that can be used by the scene manager to be put into the scene. This includes objects, cameras, lights, sky boxes or terrains. Movable objects can have a parent. If so the local transform is relative to the parent transform. This relation is stored in the parent and children pointers.
